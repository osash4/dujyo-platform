## ğŸ› Reporting Bugs

### Critical Issues (Report Immediately)
- App crashes
- Cannot connect wallet
- Cannot earn tokens
- Cannot make purchases
- Security concerns

### How to Report

**Email**: bugs@dujyo.com

**Include**:
- Your beta access code
- Device/browser information
- Steps to reproduce
- Screenshots/videos if possible
- Expected vs actual behavior

**Example**:
```
Subject: [BUG] Cannot purchase listing

Beta Code: DUJYO-BETA-001
Device: Chrome on Windows 11
Steps:
1. Go to marketplace
2. Click on listing
3. Click purchase
4. Error appears: "Insufficient balance" (but I have 100 DYO)

Expected: Purchase should complete
Actual: Error message appears
```
